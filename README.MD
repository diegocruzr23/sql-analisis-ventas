# SQL – Análisis de Ventas

Proyecto personal de práctica en SQL orientado al análisis de ventas
y generación de reportes básicos usados en entornos empresariales.

## Motor
-  MySQL

## Tablas
- clientes(id, nombre)
- productos(id, nombre, precio)
- ventas(id, cliente_id, fecha, total)
- detalle_ventas(venta_id, producto_id, cantidad)

## Consultas
- Total de ventas por cliente
- Productos más vendidos
- Clientes sin compras
- Ventas por mes
